Frequently asked questions

How does Prompt Easy work?
Type a shortcut like #investigate and press space. The shortcut automatically expands into your full prompt template. It works in any text field on any website.
Which AI platforms are supported?
Prompt Easy works on ChatGPT, Claude, Perplexity, Gemini, and any other website with text inputs. It's not limited to AI platforms.
Is my data secure?
Yes. All shortcuts are stored locally by default. Cloud sync is optional and encrypted. We never access your prompt content.
Can I create custom shortcuts?
Absolutely. Create unlimited custom shortcuts with variables like {selection}, {clipboard}, {url}, and {title} to inject context automatically.
What happens if I cancel?
You keep all your shortcuts and can continue using the free plan. No data is lost when you cancel.
